# 🛡️ FraudShield AI

FraudShield AI is an AI-powered web application that helps users detect online scams quickly and easily. It analyzes suspicious URLs, QR codes, and messages, then uses AI to determine whether they are safe or potentially fraudulent.

## 🚀 Features

- 🔗 URL Scam Detection
- 📱 QR Code Scam Analysis
- 💬 Suspicious Message Detection
- 🤖 AI-powered explanation for every result
- ⚡ Fast and user-friendly interface

## 🧠 How It Works

1. Enter or upload a URL, QR code, or message.
2. FraudShield AI analyzes the input.
3. The AI determines whether it is safe or suspicious.
4. The app provides a detailed reason behind its decision, helping users understand the potential risks.

## 🛠️ Technologies Used

- React
- TypeScript
- Vite
- Tailwind CSS
- AI API Integration

## 🎯 Purpose

The goal of FraudShield AI is to help people stay safe from online scams by providing instant AI-powered scam detection and clear explanations.
